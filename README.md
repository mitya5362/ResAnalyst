# База данных + Python-скрипты + Численные методы (C++) + Статистический анализ (Excel)

## Общее описание

Проект включает четыре самостоятельные работы:

1. **Проектирование и анализ реляционной базы данных** (PostgreSQL)
2. **Автоматизация и обработка данных на Python**
3. **Численное решение систем линейных уравнений** (C++)
4. **Статистический анализ выборки** (Excel / математическая статистика)

---

## Часть 1. База данных

### Цель
Спроектировать структуру базы данных для предметной области, наполнить тестовыми данными и выполнить сложные аналитические запросы с оценкой их производительности.

### Реализовано
- **Схема БД** (15+ таблиц) с нормализацией, первичными и внешними ключами, ограничениями
- **Наполнение данными** — генерация тестовых записей через `INSERT`
- **Аналитические запросы** — группировки, агрегации, подзапросы, оконные функции, рекурсивные CTE
- **Оптимизация** — сравнение разных способов решения одной задачи
- **Анализ планов выполнения** — `EXPLAIN ANALYZE`, оценка стоимости, типы соединений

---

## Часть 2. Python-скрипты

### Цель
Автоматизировать взаимодействие с базой данных, обработать результаты запросов и визуализировать данные.

### Реализовано
- **Подключение к PostgreSQL** через `psycopg2`
- **Автоматическое выполнение SQL-запросов** из Python-скриптов
- **Генерация тестовых данных** — заполнение БД случайными значениями
- **Экспорт результатов** в CSV и Excel
- **Визуализация** — графики погрешностей, сравнение времени выполнения запросов (matplotlib)
- **Пакетная обработка** — запуск серии запросов и сохранение планов выполнения

### Библиотеки
`psycopg2`, `pandas`, `matplotlib`, `numpy`, `openpyxl`

### Что показывает
- Умение интегрировать Python с базами данных
- Навыки автоматизации и обработки данных
- Визуализацию результатов для анализа

---

## Часть 3. Численные методы (C++)

### Цель
Реализовать алгоритм решения системы линейных уравнений с трёхдиагональной матрицей (метод прогонки) и исследовать его точность и устойчивость.

### Реализовано
- **Класс для работы с векторами** — перегрузка операторов, ввод/вывод, генерация случайных значений
- **Класс для трёхдиагональных матриц** — хранение трёх диагоналей, умножение на вектор, решение СЛАУ
- **Метод прогонки** — прямой и обратный ход с контролем вычислительной устойчивости
- **Тестирование** — серия экспериментов для матриц разного размера и обусловленности
- **Оценка погрешности** — вычисление абсолютной и относительной ошибки

---

## Часть 4. Статистический анализ выборки (Excel)

### Цель
Выполнить полный статистический анализ числовой выборки: построение вариационного ряда, интервального распределения, расчёт основных статистик и доверительных интервалов.

### Реализовано
- **Вариационный ряд** — сортировка исходных данных
- **Интервальное распределение** — определение числа интервалов (формула Старджесса), шага разбиения, границ, частот
- **Основные статистики** — среднее, дисперсия, СКО, медиана, мода, коэффициент вариации, размах
- **Оценка однородности выборки** — по коэффициенту вариации
- **Доверительные интервалы** — для генеральной средней и доли (с повторным и бесповторным отбором)
- **Расчёт оптимального объёма выборки**

### Файл
`ТерВер.xlsx` — содержит все расчёты с формулами

### Что показывает
- Понимание основ математической статистики
- Умение работать с распределениями и интервальными оценками
- Навыки аккуратных расчётов в табличном процессоре
