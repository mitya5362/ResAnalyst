# База данных + Python

## Назначение
Проектирование реляционной базы данных для гостиничного комплекса, наполнение тестовыми данными, выполнение аналитических SQL-запросов и автоматизация обработки результатов на Python.

## Технологии
| Компонент | Технологии |
|-----------|------------|
| СУБД | PostgreSQL |
| Язык запросов | SQL (оконные функции, CTE, рекурсивные запросы) |
| Python | psycopg2, pandas, matplotlib, numpy |

## Реализовано

### 1. Моделирование БД
- Схема из 15+ таблиц (гости, отели, номера, владельцы, услуги, штрафы, проживания и др.)
- Первичные и внешние ключи, ограничения целостности
- Индексы для оптимизации запросов

### 2. SQL-запросы (50+)
- Агрегации и группировки (`GROUP BY`, `HAVING`)
- Подзапросы и `EXISTS` / `NOT EXISTS`
- Оконные функции (`RANK`, `DENSE_RANK`, `ROW_NUMBER`, `LAG`)
- Рекурсивные `CTE`
- `UNION` / `UNION ALL`
- Сравнение 2-3 способов решения одной задачи

### 3. Анализ производительности
- `EXPLAIN ANALYZE` для каждого запроса
- Сравнение планов выполнения (Hash Join, Nested Loop, Merge Join)
- Анализ стоимости, времени выполнения, использования буферов
- Оценка эффективности индексов

### 4. Python-скрипты
- Подключение к PostgreSQL через `psycopg2`
- Автоматическое выполнение SQL-запросов
- Генерация тестовых данных
- Экспорт результатов в CSV / Excel
- Визуализация графиков (matplotlib)
