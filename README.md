# Описательная статистика и доверительные интервалы

## 📌 О чём проект
- расчёт основных статистик выборки (среднее, дисперсия, медиана, мода, коэффициент вариации);
- построение вариационного ряда и интервального распределения;
- доверительные интервалы для генеральной средней и доли (с повторным и бесповторным отбором);
- оценку однородности выборки.

## 🧠 Что демонстрируют расчёты
- Навыки работы с **выборками** и **распределениями**.
- Построение **интервального ряда** (границы, частоты, высоты).
- Вычисление ключевых статистик: среднее, дисперсия, СКО, медиана, мода.
- Проверку **однородности выборки** через коэффициент вариации.
- **Доверительные интервалы** (генеральная средняя, генеральная доля).
- Расчёт **оптимального объёма выборки** (с повторениями и без).

## 📁 Структура файла
| Область | Содержание |
|--------|------------|
| A..B | исходные наблюдения (`Значение Х`) |
| C | вариационный ряд |
| F..G | границы интервалов группировки |
| H..J | частоты, высоты, средние значения интервалов |
| L | выборочные статистики (среднее, дисперсия, медиана, вариация) |
| L..N | доверительные интервалы + расчёт оптимального объёма выборки |
